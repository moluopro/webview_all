`webview_all` 是一个Flutter WebView 库，支持 Android、iOS、macOS、Windows、Linux、OpenHarmony 和 Web 等平台，遵守 [webview_flutter 平台接口](https://pub.dev/packages/webview_flutter_platform_interface)。

## 平台兼容性

| 平台 | 最低要求 | 引擎 |
| --- | --- | --- |
| Android | API 24+ | `WebView` |
| iOS | 13.0+ | `WKWebView` |
| macOS | 10.15+ | `WKWebView` |
| Windows | Windows 10 1809+ | `WebView2` |
| Linux | `webkit2gtk-4.1` | `WebKitGTK` |
| OHOS | API 12+ | `ArkWeb` |
| Web | 无 | `JS interop` |
